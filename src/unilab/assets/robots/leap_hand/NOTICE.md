# LEAP Hand V1 model assets

The robot geometry in this directory was converted from the LEAP Hand V1 URDF
in [leap-hand/LEAP_Hand_Sim](https://github.com/leap-hand/LEAP_Hand_Sim),
retrieved from the repository's `master` branch on 2026-07-12.

`LICENSE.txt`.

The generated `leap_hand.xml` is the robot-level MJCF. Task objects, scene
geometry, and starting poses belong in task-level fragment XML files such as
`scene.xml`.
